# Love Counter Website (PN & QL)

Một trang web kỷ niệm tình yêu dành cho Phúc & Lưu.

## Tính năng
- **Love Counter**: Đếm ngày yêu nhau.
- **Timeline**: Các cột mốc đáng nhớ.
- **Music Player**: Playlist nhạc lãng mạn.
- **Thử thách**: Nút "Không" chạy trốn cực gắt.

## Cài đặt
1. Clone repo.
2. Mở `index.html`.
